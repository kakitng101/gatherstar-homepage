How to track conversions with Voluum

Step 1: Create the Goal ID for the Postback URL

On the Advertiser account, go to Campaigns > Conversion Tracking where you will be able to create the Goal ID and get the structure to be used in the postback URL:

Follow the Goal creation process mentioned here.

Step 2: Add your network as traffic source in Voluum and configure the Postback

To track conversions on the Ad Server you will need to add a dynamic tracker to your campaign URL {conversions_tracking} which identifies each click uniquely.

You must store this tracker on Voluum and get it to notify your Ad Server each time a user generates a conversion. This is achieved in Voluum by configuring your network as the traffic source, using the pre-defined template.

Click the “Traffic sources” tab, then click on "Create" > "Traffic Source".

You can then follow the instructions provided by Voluum here.

When you get to the Postback URL field, replace the word “REPLACE” with the goal ID value you generated in STEP 1.

Your postback URL should now look like the following:

http://s.gatherstar.io/tag.php?goal=66a8605e6cce49fbb8056f273f8e1a2e&tag={externalid}

Note: If you selected Dynamic value when setting up your Goal, then you will also have to add &value={payout} at the end of the Postback URL so as to pass the value of the conversion. In this case, your postback URL will look like the following:

http://s.gatherstar.io/tag.php?goal=66a8605e6cce49fbb8056f273f8e1a2e&tag={externalid}&value={payout}

Once you have made these edits, you can go ahead and save the Postback.

Note: s.gatherstar.io to be substituted with the conversion domain used by your ad server.

Step 3: Create an Offer in Voluum

Click the ”Offers” tab, then click on "Create" > "Offer" .

Select the Affiliate network (if any), the Country label and put a Name to the Offer.

Then, add the URL of your product to the Offer URL. For instance:

http://mypage.com/?a=14030&c=64375&aff_id=2422&s1=

In the Conversion tracking section from the offer, under "Payout", you can select "Auto" or "Manual". If you added an affiliate network in the previous step, then you can select "Auto" so that the value is read from your affiliate network payout token. Make sure that your affiliate network is connected to your Voluum account correctly.

If you did not add an Affiliate network to your Voluum account or the network does not pass any payouts dynamically, then select "Manual" and add the value of the payout.

Note: Remember that the Postback URL you see here corresponds to the Affiliate Platform. This is not the same Postback URL as the one you used in Traffic Source settings.

Click "Save" to Save the offer.

Step 4: Create a campaign in Voluum

Click the "Campaigns" tab. When you click on "Create", you will have two options "Campaign Simple" and "Campaign Advanced". In this guide, we will use "Campaign Simple".

In the "General" settings, select the Custom Traffic Source you setup on Step 2 and add a "Name" to your campaign.

Select the "Cost Model" from your campaign and the "Cost Value" (note that these parameters will be overwritten if you enable API).

Complete the other fields until you reach "Campaign Destination". In "Campaign Destination", select the offer you want to promote and determine whether you will use any landing pages for this offer.

Click "Save". Voluum will show your Campaign URL. You will need this URL to set up your campaign in your Ad Server.

Example campaign URL:

https://voluum-domain.com/bc286faf-3176-4e88-a398-428a824c051c?campid={campaign_id}&varid={variation_id}&source={src_hostname}&keyword={keyword}&tags={tags}&siteid={site_id}&zoneid={zone_id}&catid={category_id}&country={country}&format={format}&cost={actual_cost}&tag={conversions_tracking}

Step 5: Create the campaign on GatherStar

Paste the campaign URL created in the previous step as your "Variation" for your campaign. Make sure it contains the {conversions_tracking} tag to send the conversion details to Voluum.

Finish the campaign set up. You can find a detailed guide on how to create a campaign here.

If the set up has been done correctly, you will start seeing conversions both in Voluum and your ad server. Do not forget to enable the corresponding Goal column in your Campaigns List and your Statistics.
