Response

Note: The response is made in JSON format.

Response Parameters

Type

In the response, the "type" field sets what ad format is being used. It can be any one of the following:

  • banner
  • sticky_banner
  • popunder
  • instant_message
  • mobile_instant_message
  • native_ad
  • outstream_video

 

Impression Tracking

In order to record impressions for an ad, the response contains a field called "impressions". This field contains a URL, unique to that ad, that records when an impression takes place.

The best way to use this impression URL is to GET it in the user's browser.

One way to achieve this is to:

  • Add <img src="IMPRESSION_TRACKER_URL" width="0" height="0"> to the page where the ad will be displayed.

    • The IMPRESSION_TRACKER_URL is the URL returned in the "impression" field of the response.

Outstream Video

Zones

ParameterDescription
idzoneThe ID of the ad zone.
typeThe ad format used, e.g. outstream_video.

 

Data

ParameterDescription
urlThe URL that the ad directs to.
impressionThe URL of the impression tracker.
trackingAn object with tracking info.
- offsetProgress marker to persist the video view.
- urlURL to call to persist the video view.
videoThe video ad file to load.
brandingEnabledFlag to enable the branding.
frequencyPeriodHow often in minutes the ad shows.
maximumWidthVideo width size.
impressionThe URL of the impression tracker.
